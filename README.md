# TAPES — Empirical Analysis of Global Terrorism Patterns

**Independent Research Project | Data-Driven Strategy & Insights | Oct – Nov 2025**

## Live Project
[View Interactive Dashboard](https://tapes-eight.vercel.app/)

## Overview
End-to-end independent research project analyzing 10+ years of global terrorism 
records to surface regional trends, temporal patterns, and operational insights 
for policy decision-making. Managed all project phases independently — from 
scoping and data collection to final executive-ready delivery.

## Key Outputs
| Deliverable | Detail |
|-------------|--------|
| Dataset Scope | 10+ years of global terrorism records |
| Analysis Type | Regional trends, temporal patterns, operational insights |
| Final Output | Executive-ready report with data visualizations |
| Framework Used | MECE (Mutually Exclusive, Collectively Exhaustive) |

## Project Phases
1. **Scoping** — Defined research objectives and analytical framework
2. **Data Collection** — Sourced and cleaned large-scale historical incident data
3. **Analysis** — Applied MECE framework to identify patterns and trends
4. **Reporting** — Translated findings into structured strategic insights
5. **Delivery** — Published interactive dashboard for stakeholder consumption
